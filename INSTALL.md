# 安装说明

社交媒体内容雷达支持 Chrome、Edge、Arc、Brave 等 Chromium 内核浏览器。

## 推荐方式：官网下载

访问官网下载安装页：

[https://smcr.top/download.html](https://smcr.top/download.html)

根据页面提示选择应用商店安装或离线安装。

## Chrome 应用商店

如果你可以正常访问 Chrome 应用商店，建议优先使用商店安装。商店安装可以自动更新，安全性和稳定性更好。

Chrome 应用商店地址请以官网下载安装页展示为准：

[https://smcr.top/download.html](https://smcr.top/download.html)

## 离线安装

如果你无法访问 Chrome 应用商店，可以使用离线安装方式。

1. 前往官网下载安装包。
2. 解压安装包。
3. 打开浏览器扩展页面：
   - Chrome：`chrome://extensions`
   - Edge：`edge://extensions`
4. 打开右上角“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择解压后的插件文件夹。
7. 打开小红书或抖音页面，点击插件图标开始使用。

## 注意事项

- 离线安装版本不会自动更新，需要根据官网提示手动更新。
- 解压后的插件文件夹不要删除，否则浏览器会找不到插件文件。
- 插件需要在你自己登录的小红书或抖音页面中使用。
- 若遇到问题，请先查看使用手册：[飞书文档](https://eqmw9iiyfbw.feishu.cn/docx/ZCXhddvWvoCuVqx5hVvcBnNsnZT)。
